# 4周作战计划：50单手动撮合 + H4验证

> 版本：v1.0 | 日期：2026-02-07
> 核心目标：4周内完成50单手动撮合，验证H4（素材跨账户跑量效果是否可迁移）
> 判断标准：跨账户eCPM衰减<=30%为Go，>50%为致命信号

### 创作者招募方案

**目标：50个长尾创作者，>=30个意向**

**筛选标准：**
- 粉丝量：1万-10万（长尾，变现难，意愿高）
- 发布频率：近30天>=8条（活跃，有素材库存）
- 品类：美妆/食品/3C（匹配商家需求）
- 内容有商业投放价值（非纯娱乐、有产品露出或使用场景）

**触达渠道（按优先级）：**

| 渠道 | 预计触达量 | 转化率预估 | 负责人 |
|------|-----------|-----------|--------|
| 抖音私信（直接联系） | 30人 | 30-40% | OP |
| MCN朋友圈/行业群 | 10人 | 50-60% | OP |
| 小红书/微博交叉定位 | 5人 | 20-30% | OP |
| 创作者服务平台（蒲公英等） | 5人 | 20-30% | OP |

**沟通话术（模板）：**

> **首次触达（抖音私信/微信）：**
>
> 你好！我是[名字]，在做一个短视频素材推荐服务。
>
> 注意到你之前发的[具体视频标题/内容描述]数据很不错。这条内容现在流量高峰过了，但素材本身的商业价值还在——有品牌方愿意买来投千川。
>
> 简单说就是：你的老内容，不用再做任何事情，授权给商家就能赚一笔。根据你这条素材的数据，预计能卖[X]元。
>
> 有兴趣聊聊吗？可以随时问我细节。

> **深入沟通（已表示兴趣后）：**
>
> 具体流程很简单：
> 1. 你选想授权的视频（建议发布超过8天、流量已过峰的）
> 2. 签一个授权协议（授权范围仅限千川投放，不改内容，到期自动失效）
> 3. 有商家匹配上就成交，钱直接打到你账户
>
> 首批合作伙伴我们有保底：即使没人买，上架满30天也有[X]元保底收益。
>
> 你手上有哪些老视频觉得可以拿出来试试？

### 撮合SOP（从发现匹配到完成交易）

```
Step 1: 匹配识别（OP，<1小时）
├── 输入：商家需求（品类+预算+内容偏好）+ 素材库
├── 动作：在素材库中筛选3-5条候选素材
├── 输出：匹配方案（含素材卡片+价格+创作者信息）
└── 工具：飞书多维表格筛选

Step 2: 商家确认（OP，<24小时）
├── 输入：匹配方案
├── 动作：微信/飞书发送素材卡片给商家，答疑
├── 输出：商家选择意向素材+确认价格
├── 记录：商家反馈（选了哪条、没选的原因、价格异议）
└── 工具：微信/飞书 + 飞书表格记录

Step 3: 创作者确认（OP，<24小时）
├── 输入：商家确认的素材+成交价格
├── 动作：联系创作者确认授权意愿和价格分成
├── 输出：创作者同意/拒绝
├── 记录：创作者反馈（价格满意度、授权顾虑）
└── 如拒绝：记录原因，返回Step 1重新匹配

Step 4: 合同签署（OP+LG，<24小时）
├── 输入：双方确认的交易条款
├── 动作：填写合同模板，双方电子签名
├── 输出：已签署的三方协议（创作者-平台-商家）
├── 合同要素：素材ID、授权范围（仅千川投放）、授权期限（30/60/90天）、
│            价格、付款方式、肖像权声明、违约条款
└── 工具：石墨文档/腾讯电子签

Step 5: 付款+素材交付（PM+OP，<24小时）
├── 输入：已签署合同
├── 动作：商家付款（微信/支付宝转账） → 平台确认 → 创作者发送素材源文件
├── 输出：交易完成确认
├── 记录：付款时间、金额、素材交付确认
└── 资金流：商家→平台账户→扣除平台佣金20%→创作者

Step 6: 投放跟踪启动（DA，交付后立即）
├── 输入：已交付素材的商家信息
├── 动作：建立投放跟踪卡片，约定商家回传数据周期
├── 输出：跟踪卡片+商家承诺的数据回传时间
└── 工具：飞书多维表格-投放跟踪表
```

**全流程时间目标：从匹配到交付<=48小时**

| 转化量 | 数字 | 下单/加购/表单提交等 |
| eCPM | 数字 | 千次展示收入（元），=消耗/展示×1000 |
| CTR | 数字 | 点击率，=点击/展示 |

| 备注 | 文本 | 异常情况说明 |

### 关键指标定义和计算方法

| 指标 | 定义 | 计算方法 | 通过标准 |
|------|------|----------|----------|
| **跨账户eCPM衰减率** | 素材从原账户到新账户的eCPM下降幅度 | (原eCPM - 新eCPM) / 原eCPM × 100% | <=30% Go, >50% 致命 |
| **跑量成功率** | 素材在新账户达到有效消耗的比例 | 7天累计消耗>=500元的素材数 / 总投放素材数 | >=自制素材1.2倍 |
| **采购vs自制对比** | 同一商家采购素材与自制素材的效果比 | 采购素材平均eCPM / 自制素材平均eCPM | >=1.0 |
| **价格交集覆盖率** | 买卖双方可接受价格的重叠比例 | 成交价在双方可接受区间内的交易数 / 总交易数 | >=70% |
| **创作者上架率** | 触达后愿意上架的比例 | 同意上架人数 / 总触达人数 | >=60% |
| **商家付费转化率** | 展示后愿意付费的比例 | 成交商家数 / 展示商家数 | >=48% |
| **交易完成时长** | 从匹配到素材交付的总时间 | 交付时间 - 首次匹配时间 | <=48小时 |

| **应对C：认知问题** | 制作1分钟说明视频，用成功案例说明（如："XX创作者靠3条老视频多赚了XX元"） |
| **应对D：人群错误** | 从"长尾个人创作者"转向"MCN旗下的小号/过气达人"，或直接与2-3家MCN合作批量获取素材 |
| **最坏情况** | W1结束意向<15人 → 转为MCN合作模式，直接与MCN谈批量授权 |

### 风险3：平台限流/封杀

| 维度 | 内容 |
|------|------|
| **预警信号** | (1)买来的素材在千川投放被限流 (2)收到字节/千川的警告通知 (3)ISV调研发现明确禁止条款 |
| **应对A：投放限流** | 分析限流原因（是素材本身还是账户问题），如果是素材跨账户的系统限制，这本身就是H4的答案——不可迁移 |
| **应对B：官方警告** | 立即停止相关操作，评估警告严重程度，如果是"不建议"可调整方式继续；如果是"禁止"则触发No-Go |
| **应对C：ISV禁止** | 放弃ISV路径，评估MCN+平台模式是否可行，天花板降低但可能仍有空间 |
| **预防措施** | (1)W1-W2期间不做任何自动化/批量操作 (2)所有操作由商家在自己的千川后台手动完成 (3)不碰任何API (4)PM通过行业人脉非正式探测字节态度 |
| **最坏情况** | 确认字节会封杀 → 评估快手/小红书跨平台可行性，或直接止损 |

### 风险4：数据不够（W4时投放数据覆盖<50%）

| 维度 | 内容 |
|------|------|
| **预警信号** | W3中期报告时有投放数据的素材<10条 |
| **根因诊断** | (1)商家买了不投 (2)商家投了不给数据 (3)成交太晚投放周期不够 |
| **应对A：买了不投** | OP逐一电话跟进，了解不投原因；提供投放指南和出价建议；必要时提供小额投放补贴（如报销首100元消耗） |
| **应对B：不给数据** | (1)承诺数据仅用于分析不外传 (2)下次交易给10%折扣换数据 (3)DA现场协助导出 |
| **应对C：投放周期不够** | W2尽量前置成交（D8-D10集中成交），确保到W4有>=14天投放数据；W4可延长3-5天收集数据再做Gate决策 |
| **最坏情况** | 数据仍不够 → Gate 1延后1周，但不超过D35做最终决策 |

### 风险5：团队执行力不足

| 维度 | 内容 |
|------|------|
| **预警信号** | 连续3天未完成每日目标 |
| **应对** | (1)PM立即1v1沟通了解原因 (2)如果是任务量过大，调整分工或引入临时帮手 (3)如果是能力不匹配，快速换人 |
| **预防措施** | (1)每日站会跟踪进度 (2)关键任务有明确的daily target (3)W1结束后评估团队配合度 |
