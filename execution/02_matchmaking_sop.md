> 产品经理交付 | 2026-02-07

> 任务目标：在前4周完成5-10笔真实交易，验证H1-H4核心假设

## 1. 撮合流程图

### 完整流程总览（预计单笔5-7天）

```
[第0天] 发现商家需求
   ↓ (运营，微信/飞书，30分钟，需求登记表)
[第0-1天] 商家需求收集与评分
   ↓ (运营，电话/微信，45分钟，需求评分表)
[第1-2天] 匹配候选创作者内容
   ↓ (运营，飞书表格，2小时，候选清单)
[第2-3天] 向商家推荐素材
   ↓ (运营，微信，1小时，素材推荐话术)
[第3-4天] 商家确认购买意向
   ↓ (运营+商家，微信，30分钟，购买确认书)
[第4天] 联系创作者谈授权
   ↓ (运营，微信，1小时，授权协议模板)
[第4-5天] 创作者确认授权
   ↓ (创作者，微信/飞书，30分钟，签署协议)
[第5天] 商家付款
   ↓ (商家→公司，对公转账/微信支付，即时，付款凭证)
[第5天] 平台打款给创作者
   ↓ (公司→创作者，微信/支付宝，即时，转账记录)
[第5天] 素材交付
   ↓ (创作者→运营→商家，网盘链接，1小时，源文件+使用说明)
[第5天+] 商家投放
   ↓ (商家，千川后台，自主操作)
[第6天] 投放跟进（D+1）
   ↓ (运营，微信，15分钟，数据采集表)
[第8天] 投放跟进（D+3）
   ↓ (运营，微信，15分钟，数据采集表)
[第12天] 投放跟进（D+7）
   ↓ (运营，微信，30分钟，完整效果复盘)
[第12天] 交易完成，数据归档
   ↓ (运营，飞书表格，30分钟，交易档案)
```

### 2.2 电话沟通脚本（45分钟标准流程）

**【开场白】（2分钟）**
```
您好，我是XXX（项目名称）的XX，我们是专门帮千川投手解决素材焦虑的。
看到您在XX（来源渠道）上发过求素材的信息/您是XX品牌的投手对吗？
现在方便聊几分钟吗？我想了解一下您在千川投放上的素材需求。
```

**【现状探查】（10分钟）**
- **问题1：** 您现在一周大概能产出多少条新素材？
  - （听完回答）那这些素材里，真正能跑起量的大概有几条？
  - **追问：** 所以其实有X条的缺口对吗？这X条现在怎么补？

- **问题2：** 您测素材的成本大概多少？
  - （如果不清楚）我是说，拍一条+投放测试，不能跑的话，大概浪费多少钱？
  - **记录：** 测试成本=_____元，用于后续定价参考

- **问题3：** 您现在最缺什么样的素材？
  - （具体追问风格/品类/时长，填入需求表"素材偏好"）
  - **不要问：** "您对素材有什么要求？"（太空泛）

**【痛点确认】（5分钟）**
- **问题4：** 如果有一条素材，是其他商家已经在千川投过、跑出过量，您愿意买吗？
  - （如果犹豫）您担心的是什么？价格？还是怕跑不动？
  - **记录：** 核心顾虑=________，后续话术针对性解决

- **问题5：** 您觉得这样的素材，多少钱您能接受？
  - （不要先报价，先听商家出价）
  - **记录：** 心理价位=_____元

**【价值传递】（5分钟）**
```
明白了。我们做的就是这件事：
我们从达人那里拿到他们"已经涨粉见顶"的旧视频，这些视频是真实跑过量的，
有播放数据、互动数据，甚至有些是达人自己挂过小黄车卖货的。

我们会把数据给到您，您可以先看数据、看预览，觉得合适再买。
买了之后，您拿到的是源文件+完整授权合同，可以直接投千川，合规无风险。

我们已经帮XX品牌（如果有case）拿到过X条素材，他们测下来跑量成功率比自己拍的高X倍。
```

**【需求确认】（3分钟）**
- **问题6：** 您现在急需几条素材？什么时候要？
  - **记录：** 紧急度=_____
- **问题7：** 我这边给您匹配几条候选素材，明后天能给您看，您方便吗？
  - **约定下次沟通时间**

**【结尾】（2分钟）**
```
好的，那我记录一下：
您是XX品类，月消耗XX万，现在缺XX条素材，愿付价XX-XX元，X天内要。
我这边明后天给您整理XX条候选素材，带数据，微信发给您。
到时候您看完觉得哪条合适，咱们再聊具体怎么买。
加您微信可以吗？
```

**【不要问的问题】**
- ❌ "您对我们平台有什么建议？"（现在还没有平台）
- ❌ "您愿意先付定金吗？"（吓跑商家）
- ❌ "您能接受效果分成吗？"（Phase 1不做分成）
- ❌ "您需要AI换人吗？"（Phase 1不做）

### 2.3 需求优先级评分方法（运营内部评分表）

每个商家需求完成收集后,运营填写评分表,决定优先服务顺序。

| 评分维度 | 权重 | 评分规则（0-10分） | 得分 |
|---------|------|------------------|------|
| **紧急度** | 30% | 今天要=10分，3天内=7分，本周内=5分，不急=2分 | ___ |
| **品类匹配度** | 25% | 美妆/食品/3C=10分（MVP重点），其他=5分 | ___ |
| **付费能力** | 20% | 愿付≥1000元=10分，500-1000=7分，<500=4分 | ___ |
| **决策效率** | 15% | 投手自己拍板=10分，需主管=7分，需老板=3分 | ___ |
| **验证价值** | 10% | 月消耗≥10万=10分（H2/H4验证价值高），<10万=5分 | ___ |

**总分=各维度得分×权重，优先服务总分≥7分的商家。**

## 3. 内容匹配规则

### 3.1 人工匹配标准（运营操作手册）

**步骤1：从创作者素材库中筛选候选内容**

打开【创作者素材库】飞书表格,按以下条件筛选：

1. **品类强匹配**
   - 商家品类=美妆 → 筛选创作者内容标签包含"美妆""护肤""彩妆"
   - 商家品类=食品 → 筛选"零食""饮品""食品测评"
   - 商家品类=3C → 筛选"数码测评""手机配件""电子产品"

2. **数据门槛**
   - 播放量：≥5万（美妆/3C）或≥2万（食品）
   - 点赞率：≥2%
   - 评论数：≥100条
   - 内容状态：发布≥8天（流量峰值已过）

3. **风格匹配**
   - 对照商家需求表的"素材偏好"
   - 如商家要"达人出镜",排除纯产品展示类
   - 如商家要"15秒内",排除长视频

**步骤2：质量人工复审（每条素材30秒判断）**

打开抖音查看候选素材原视频，判断：

| 检查项 | 判断标准 | 不通过则排除 |
|-------|---------|------------|
| 内容完整性 | 视频有头有尾，不是二次剪辑片段 | ✓ |
| 广告适配性 | 可以无缝替换产品（如开箱、测评结构清晰） | ✓ |
| 达人形象 | 出镜者形象得体，符合品类调性 | ✓ |
| 平台合规 | 无违禁词、极限词、虚假宣传 | ✓ |
| 音乐版权 | 使用抖音曲库音乐，非商业独家版权音乐 | ✓ |

**步骤3：生成推荐清单**

每个商家需求推荐3-5条候选素材（不要太多，选择困难）。

### 3.2 匹配质量评分（1-5星，内部记录）

运营完成匹配后，对每条素材进行质量评分，用于后续优化匹配规则。

| 评分 | 标准 | 说明 |
|------|------|------|
| ⭐⭐⭐⭐⭐ | 完美匹配 | 品类/风格/数据全对，商家看到大概率立刻买 |
| ⭐⭐⭐⭐ | 优质匹配 | 品类对，数据好，风格有小偏差 |
| ⭐⭐⭐ | 合格匹配 | 品类对，数据达标，但亮点不强 |
| ⭐⭐ | 勉强匹配 | 品类边缘相关，数据一般 |
| ⭐ | 凑数匹配 | 库存不足时的无奈选择 |

**目标：推荐给商家的素材平均≥4星。如果库存不足无法达到，说明供给端需要紧急扩充。**

### 3.3 推荐给商家时的展示话术

**微信文案模板：**

```
【商家名称】您好，根据您的需求（XX品类，XX风格），我这边筛选了X条候选素材，都是真实跑过量的：

   预览：（抖音链接）
   价格：1200元

📹 素材2：【简短描述】
   数据：播放8.7万 | 点赞1850 | 评论298
   适合：【匹配点】
   预览：（抖音链接）
   价格：800元

📹 素材3：【简短描述】
   数据：播放6.3万 | 点赞1520 | 评论412
   适合：【匹配点】
   预览：（抖音链接）
   价格：900元

```

**话术要点：**
- ✅ 用数据说话（播放/点赞/评论）
- ✅ 明确价格（降低询价摩擦）
- ✅ 可点击预览（降低决策门槛）
- ✅ 强调"真实跑过量"（核心卖点）
- ❌ 不要说"这是旧视频"（降价感）
- ❌ 不要说"达人不要了"（贬低素材）
- ❌ 不要过度承诺"肯定能跑量"（规避纠纷）

### 4.3 付费流程

**场景1：商家对公转账（可开发票）**

1. 商家转账后，截图给运营
2. 运营确认到账（通常T+1工作日）
3. 到账后，运营联系财务开具发票
4. 发票内容：技术服务费 / 信息服务费（不要写"视频购买"）
5. 发票快递给商家

**场景2：商家微信/支付宝转账（不开票）**

1. 商家扫码转账，即时到账
2. 运营确认收款截图
3. 无需开票

**平台打款给创作者（统一流程）**

1. 商家付款到账后，运营在【交易跟踪表】更新状态为"已付款"
2. 运营联系创作者："XX商家已付款，确认授权后24小时内打款给您"
3. 创作者在电子签平台签署协议
4. 协议签署完成后，运营提交【创作者付款申请单】（飞书审批流）
   - 收款人：创作者姓名
   - 收款账户：微信/支付宝
   - 金额：____元（成交价×80%）
   - 附件：协议PDF + 商家付款截图
5. 财务审批通过后，24小时内转账
6. 转账完成后，运营截图给创作者确认

## 6. 异常处理

### 6.1 商家买了不投怎么办

**场景：** 商家付款、拿到素材后，超过7天未投放。

**处理流程：**

1. **D+3主动询问**

```
【商家名称】您好，素材交付3天了，方便问下投放了吗？
如果遇到什么问题可以跟我说~
```

2. **D+7再次询问**

```
【商家名称】您好，素材拿到一周了还没投吗？
是遇到什么问题了吗？还是暂时不急用？

如果暂时不投的话也没关系，授权期是12个月，您什么时候投都可以。
就是我们这边要做个数据跟踪，想确认一下~
```

3. **D+14最终确认**

```
【商家名称】您好，再确认一下，素材还投吗？

如果确定不投了，麻烦告诉我原因：
1️⃣ 临时不需要了（品类/策略调整）
2️⃣ 审核没通过
3️⃣ 觉得素材不合适
4️⃣ 其他原因

您的反馈对我们很重要，不会影响后续合作~
```

4. **内部记录**

【交易跟踪表】备注："商家未投放，原因：________"
计入H2验证的"无效样本"（不纳入跑量成功率计算）

**是否退款：**
- **不退款。** 协议中明确"授权期12个月"，商家已获得使用权。
- 如果商家强烈要求退款且理由合理（如"审核不通过""素材严重不符"），可个案处理，但需报项目负责人审批。

### 6.2 投了效果很差要退款怎么办

**场景：** 商家投放后，ROI远低于预期，要求退款。

**处理流程：**

1. **先收集数据**

```
【商家名称】您好，我理解您的情况。
能先麻烦您给我看一下具体的投放数据吗？
（消耗、点击率、转化率、ROI）

我需要确认一下是素材问题，还是其他因素（出价/定向/商品页等）。
```

2. **分析原因**

| 情况 | 判断 | 处理 |
|------|------|------|
| 素材点击率低（<1%） | 素材hook弱，可能是素材问题 | 可协商部分退款或换素材 |
| 点击率正常但转化率低 | 商品页/价格问题，非素材问题 | 不退款，但给建议 |
| 消耗太少（<1000元） | 测试预算不足，无法判断 | 建议加大预算再测 |
| 同品类其他计划也跑不动 | 账户/品类问题，非素材问题 | 不退款 |

3. **回复话术**

**情况A：素材确实有问题（点击率<1%，远低于同品类均值）**

```
【商家名称】您好，看了数据，这条素材点击率确实偏低，可能跟您的品类/人群匹配度不够好。

我们这次是第一批验证，确实有可能存在匹配不准的情况。
这样吧，我们可以：
1️⃣ 给您退50%款项（___元）
2️⃣ 或者不退款，再免费给您推荐1-2条其他素材试试

您看哪个方案合适？
```

**情况B：点击率正常，但转化率低（非素材问题）**

```
【商家名称】您好，看了数据，素材的点击率是___，这个数据其实不差。
但转化率只有___，这个一般是商品页、价格、详情页影响比较大。

咱们协议里也说了，素材跑量效果受多因素影响（账户、商品、定向等），
我们提供的是"已验证过的素材"，但不能保证100%适用所有账户。

建议您：
1️⃣ 优化一下商品页（主图/详情/价格）
2️⃣ 调整定向人群
3️⃣ 加大预算测试（现在消耗才___元，样本太小）

如果调整后还是不行，咱们再聊后续怎么办~
```

**情况C：消耗太少（<1000元）**

```
【商家名称】您好，看了一下，这条素材才跑了___元消耗，这个量太小了，
千川系统还在冷启动阶段，数据波动会很大。

建议您至少测到2000-3000元消耗再看，不然没法判断。
我们之前有商家前500元数据很差，但1000元后突然起量了。

您要不再测测看？如果实在不想测了，咱们再讨论退款的事~
```

4. **退款政策（内部标准）**

| 情况 | 退款比例 | 审批 |
|------|---------|------|
| 素材点击率<0.5%（远低于均值），消耗>2000元 | 50% | 运营直接决定 |
| 素材严重不符（如错发素材、素材损坏） | 100% | 运营直接决定 |
| 其他情况商家强烈要求 | 个案协商 | 报项目负责人 |

**重要原则：**
- Phase 1是验证期，对商家友好，宁可退款也不要口碑崩盘
- 但要坚持"素材效果受多因素影响"的底线，不能让商家认为"跑不动就能退"
- 每一笔退款都要记录原因，用于优化匹配规则

### 6.3 创作者反悔不想卖了怎么办

**场景：** 商家已付款，但创作者突然反悔，不愿签协议/不给素材。

**处理流程：**

1. **先了解原因**

```
【创作者昵称】你好，怎么突然不想卖了呀？
方便说一下原因吗？

是担心什么问题吗？还是觉得价格低了？
咱们可以聊聊~
```

**常见原因与话术：**

| 原因 | 话术 |
|------|------|
| "我怕被盗用/滥用" | "协议里写得很清楚，只能用于千川广告，不能转售。而且商家是正规公司，有合同约束的，不会乱用。咱们已经成交XX单了，都没出过问题~" |
| "我觉得价格低了" | "咱们之前沟通的时候说的就是这个价格呀，现在商家已经付款了。如果您真的觉得亏了，我可以申请多给您加___元补偿，但合同还是要签的~" |
| "我担心粉丝看到会取关" | "这条视频已经发了__天了，流量早就没了，基本不会再有自然流量。而且商家投千川是付费流量，跟您的账号是分开的，粉丝不会因为这个取关您。" |
| "我不想签合同，太麻烦" | "合同是保护您的，证明这笔交易是合法的。在手机上签，1分钟就完成，很简单。我现在把链接发您，您点进去签个名就行~" |

2. **提价协商**

如果创作者坚持要加价，且理由合理（如确实低估了素材价值）：

```
【创作者昵称】这样吧，我申请一下，给您加___元。
但您要答应：
1️⃣ 今天签协议
2️⃣ 今天给素材
3️⃣ 后续不再改价

可以吗？
```

**加价上限：** 原价的20%以内，运营可直接决定；超过20%需报项目负责人审批。

3. **创作者坚决不卖**

```
【创作者昵称】好的，我理解。
但商家那边已经付款了，我需要给他们一个交代。

您看这样行吗：
1️⃣ 我们给商家全额退款
2️⃣ 但您这次的信用记录会被标注，后续如果再有合作机会，我们可能不会优先找您了

您确定要这样吗？
```

4. **如果创作者确定不卖**

- 给商家全额退款（当天完成）
- 给商家道歉 + 补偿：免费推荐另一条素材，或给50-100元代金券（用于下次购买）
- 【创作者库】标注该创作者"信用存疑，暂不合作"
- 如果这种情况频繁发生（>10%），说明创作者沟通和筛选环节有问题，需优化

### 6.4 发现素材被平台限流怎么办

**场景：** 商家反馈"素材投了之后被千川判定为低质内容/同质化内容，限流了"。

**处理流程：**

1. **确认限流原因**

```
【商家名称】您好，能麻烦您截图一下千川后台的提示吗？
或者在"计划诊断"里看看具体是什么问题？

可能的原因有：
1️⃣ 素材本身违规（极限词、虚假宣传）
2️⃣ 同质化（很多商家用了同一条）
3️⃣ 低质素材（点击率/转化率低）
4️⃣ 账户权重低（跟素材无关）
```

2. **问题分类处理**

| 限流原因 | 判断 | 处理 |
|---------|------|------|
| 素材违规（极限词等） | 我们审核没到位 | 道歉+退款50%+下架该素材 |
| 同质化（多商家使用） | 不可避免的风险 | 解释+建议二次剪辑+不退款 |
| 低质素材（点击率低） | 素材本身问题 | 协商退款或换素材 |
| 账户权重低 | 非素材问题 | 建议商家优化账户，不退款 |

3. **同质化问题话术**

```
【商家名称】您好，"同质化"是千川对短时间内大量相似素材的判定。

这条素材如果有多个商家同时在投，确实可能被判为同质化。
但这个问题可以通过二次剪辑规避：
1️⃣ 调整视频时长（掐头去尾）
2️⃣ 添加转场特效
3️⃣ 修改背景音乐
4️⃣ 添加品牌贴纸/文字

您要不先试试二次剪辑？如果还是不行，我们再换一条素材给您~
```

4. **内部优化**

- 如果同一条素材被多个商家投诉"同质化"，【素材库】标注该素材"高风险-同质化"
- 后续推荐时，同一条素材最多卖给3个商家
- 或者引导商家"买模板+二次剪辑"而非"买成品"
