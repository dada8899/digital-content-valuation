# 商家/投手招募执行方案

> 制定日期：2026-02-07
> 执行周期：第1-4周（手动撮合验证阶段）
> 目标：招募25个商家参与首批试点，完成5-10笔真实交易

---

## 1. 目标商家画像

### 第一批试点商家精确画像

**核心画像：**

| 维度 | 要求 |
|------|------|
| 行业 | 美妆（面膜/精华/底妆）、食品（零食/饮品）、3C数码（手机配件/小家电） |
| 月投放金额 | 30-150万（千川平台） |
| 团队规模 | 有专职投手1-3人，日常测素材节奏稳定 |
| 素材痛点程度 | 每周需测试30-80条新素材，跑量率<10%，有明确的"素材荒"焦虑 |
| 账户状态 | 千川投放至少3个月以上，有稳定的ROI基准线（ROAS 1.5-3.0） |
| 决策链条 | 投手有一定采购自主权（单次3000元以下不需要层层审批） |

**为什么选这个画像？**

1. **月投放30-150万是最佳甜蜜点**
   - 太小（<30万）：买素材的钱占投放预算比例太高，决策保守，不愿意试
   - 太大（>300万）：已有成熟的素材生产体系（内部团队/签约达人），对外部素材依赖低
   - 30-150万的商家：有钱买，有需求买，但还没建立完整体系，最焦虑

2. **聚焦美妆/食品/3C三个品类**
   - 这三个品类千川投放量级大（占整体投放的60%+）
   - 素材消耗速度快（生命周期7-14天），需求频次高
   - 跑量模板相对标准化，跨账户迁移成功率较高（预估衰减<40%）

3. **投手有自主权最关键**
   - 如果每次买素材都要走采购流程报老板，试点周期会拉长到不可控
   - 投手日常买素材代拍（500-3000元/条）已成习惯，新增一个采购渠道决策成本低

4. **必须有稳定ROI基准**
   - 如果商家本身投放数据就很乱（忽高忽低），无法验证"采购素材是否真的提升跑量率"
   - 需要有对照组（自制素材的baseline数据）

### 目标数量

**首批招募：25个商家（留40%冗余）**

拆解：
- Week 1：BD接触50个候选商家
- Week 2：筛选后深度沟通25个
- Week 3：最终确认试点意向15-20个
- Week 4：实际完成采购行为10-15个

**预期交易量：5-10笔（Gate 1最低标准是5笔）**

## 2. 招募渠道（按优先级排序）

### 渠道1：投手社群（优先级★★★★★）

**具体社群名单：**

| 社群 | 渠道 | 规模 | 进入方式 | 预估转化率 |
|------|------|------|----------|-----------|
| 千川投手交流群（微信） | 通过现有投手朋友引荐 | 200-500人/群 | 付费进群（399-999元）或老成员拉入 | 5-8% |
| 巨量千川官方社区（飞书） | 巨量引擎官方运营 | 活跃用户3000+ | 免费加入，需实名认证 | 3-5% |
| 生财有术"千川投放"圈子 | 付费社群 | 会员500+ | 年费2980元 | 8-12% |
| 抖音投流大本营（知识星球） | 独立运营 | 会员1200+ | 年费365元 | 4-6% |

**具体执行：**

1. Week 1前3天：通过2-3个现有投手朋友推荐进入2-3个核心微信群
2. Week 1第4-5天：在群内观察3天，了解投手当前讨论的高频痛点
3. Week 1第6-7天：私聊10-15个活跃投手，用调研名义建立联系（话术见第3节）
4. Week 2：从10-15个深度沟通对象中筛选出8-12个符合画像的商家进入试点

**具体说什么：**

- **群内不直接推销**——避免被踢，先做有价值的内容输出（如"分享一份美妆素材跑量数据分析"）
- **私聊切入点**："哥/姐，看你在群里讨论素材测试，我们在做一个试点项目，能否1v1聊10分钟，了解下你的素材采购需求？"
- **不要说**："我们有个素材平台想推广给你"（直接拒绝）

**预估转化率：5-8%**
- 接触100个投手 → 深度沟通6-8个 → 最终试点3-5个

### 渠道2：千川服务商（优先级★★★★☆）

**具体名单（巨量引擎官方认证服务商）：**

1. 星辰传媒（北京）
2. 量子云（杭州）
3. 巨星联动（广州）
4. 悦普集团（上海）
5. 闪电传媒（深圳）

**为什么找服务商：**

- 服务商手里有30-100个客户，相当于批量触达
- 服务商对新工具接受度高（他们本身就是工具的重度使用者）
- 服务商可能成为分销渠道（如果试点成功，他们会主动推给客户）

**进入方式：**

1. 巨量引擎官网查询服务商名录，找到对接人（通常是BD总监）
2. 邮件+电话组合拳："我们是巨量生态的素材工具开发团队，希望与贵司合作试点"
3. 话术核心：不是让他们买素材，而是**让他们帮忙推荐3-5个客户参与试点，成功后给服务商分润**

**具体说什么：**

对服务商："我们在做一个千川素材采购工具的试点，需要3-5个客户配合测试投放效果。如果试点成功，后续正式推广时贵司的客户通过您的推荐购买，我们给您15%的分润。"

**预估转化率：**
- 接触10个服务商 → 3个愿意推荐 → 每个推荐3-5个客户 → 总计10-15个商家联系方式 → 最终试点4-6个

### 渠道3：巨量引擎BD资源（优先级★★★☆☆）

**具体路径：**

1. 巨量引擎ISV合作通道（如果团队有ISV资质）
2. 通过巨量引擎的行业运营（美妆/食品/3C各有专门的行业运营团队）
3. 参加巨量引擎官方举办的"千川投放沙龙"

**进入方式：**

- 如果没有巨量BD资源：通过服务商（渠道2）引荐
- 如果有：直接对接行业运营，说明是"为千川生态开发的素材工具，需要试点商家"

**注意事项：**

- 巨量BD关心的是"你的工具能不能帮平台提升广告消耗"
- 话术重点：我们帮商家提高跑量率 → 商家愿意加大投放预算 → 平台消耗增长
- 不要说：我们做素材二次交易（平台可能担心内容版权问题）

**预估转化率：**
- 对接2-3个行业运营 → 每人推荐5-10个商家 → 总计15-30个联系方式 → 最终试点3-5个

### 渠道5：直接陌拜（优先级★☆☆☆☆）

**方式：**

1. 用"有米有数"等工具筛选出近30天千川投放活跃的美妆/食品品牌

- 建立信任周期长

**建议：只有在前4个渠道都失败的情况下才考虑。**

## 3. 招募话术

### 话术A：对投手（他关心跑量和效率）

**一句话卖点（30秒版本）：**

"哥/姐，你是不是每周要测50-100条素材，但最后能跑的只有3-5条？我们在做一个试点——卖给你'别的账户已经跑出ROAS 2.0+'的素材，你直接买了测，省掉90%的盲测。美妆素材800-1500一条，前10个试点用户首条5折。"

**详细说明（2分钟版本）：**

"我知道你现在素材获取主要靠内部团队拍+代拍+翻竞品，对吧？

这些方式有个共同问题——**跑不跑量要测了才知道，每周测试费就烧掉2-3万。**

我们在做的事是：
1. 收集那些'已经在千川投流跑出效果'的素材（ROAS 2.0+，消耗5万+）
2. 创作者授权后，你可以直接买断使用
3. 价格比你找达人拍便宜（美妆素材800-1500一条）
4. 重点是——**这条素材已经验证过能跑**，你买回去测的成功率比盲测高至少50%

现在在做小范围试点，需要10-15个投手帮忙测一下跨账户投放效果。

**试点条件：**
- 你选3-5条素材，首条5折（400-750元）
- 投放后把跑量数据给我们（不用给详细后台，就CTR/CVR/ROAS这几个核心数据）
- 如果7天内完全跑不动（消耗<1000元），我们退50%

怎么样，下周能测一条吗？"

**核心逻辑：**
- 不说"平台""市场"这种虚的词，直接说"买素材"
- 突出"已验证"这个核心价值
- 价格锚点（比达人便宜）
- 试点门槛低（首条5折+效果保障）

---

### 话术B：对商家老板（他关心成本和ROI）

**一句话卖点（30秒版本）：**

"老板，如果我说可以帮您把千川投放的测试成本降低40%，愿意试吗？我们卖的是'别人已经跑出ROAS 2.0+的素材'，您直接买了测，省掉90%的盲测费用。"

**详细说明（2分钟版本）：**

"我看您的品牌在千川投放做得挺大，我想问您一个问题——

您现在每个月花在'测试新素材但最后没跑起来'的钱，大概有多少？

我接触的商家通常是5-15万/月。这笔钱基本打水漂。

我们在做的事情，就是帮您降低这个试错成本：

**传统方式：**
- 自己拍100条素材 → 测100条 → 跑起来5条 → 浪费95条的制作费和测试费

**我们的方式：**
- 我们提供'已经在别的账户跑出ROAS 2.0+'的素材
- 您买20条测试 → 按我们预估，能跑起来10-12条（跨账户衰减30%左右）
- 跑量成功率提升1倍，测试成本直接省40%

**试点方案：**
- 第一批买5-10条，每条800-1500元（美妆）
- 我们会跟进投放数据，7天内跑不动（消耗<1000）退50%
- 如果效果好，后续按月采购

您看这个方向有兴趣吗？我安排投手跟您的团队对接一下技术细节？"

**核心逻辑：**
- 老板关心ROI和成本，用"降低试错成本40%"切入
- 用数据说话（95条浪费 vs 10-12条跑起来）
- 不要让老板直接决策细节，而是引导"投手对接投手"

### 话术C：对服务商（他关心分润和客户关系）

**一句话卖点（30秒版本）：**

"我们在做千川素材采购工具，需要3-5个客户试点。如果试点成功，后续您推荐的客户购买，我们给您15%分润。"

**详细说明（2分钟版本）：**

"XX总，我知道您服务的客户里肯定有很多在为素材发愁——每周要测几十条，跑量率不到10%。

我们在做的是一个素材采购工具：
- 把'已经在千川投流跑出效果'的素材收集起来
- 商家可以直接买断使用
- 价格比找达人拍便宜，重点是**已验证过能跑**

现在想请您帮忙：
1. 推荐3-5个客户参与试点（美妆/食品/3C品类，月投放30-150万）
2. 我们给这些客户首批素材5折优惠
3. 如果试点效果好，后续正式推广时，**您推荐的客户购买，我们给您15%分润**

对您的好处：
- 如果这个工具真的能帮客户提升跑量率，您的服务价值提升了
- 15%分润是持续的（只要客户在用，您就有收益）
- 不影响您现有的服务费模式

您看能帮忙推荐几个客户吗？"

**核心逻辑：**
- 服务商最关心"这个工具对我的客户有没有价值"和"我有什么好处"
- 分润是次要的，主要是"帮客户提升效果 = 巩固客户关系"
- 不要让服务商自己买素材（他们不是直接用户）

### FAQ应答（10个高频问题）

**Q1：这些素材别人用过了，我用还能跑吗？**

A1："会有一定衰减（我们预估30%左右），但核心逻辑是——**这条素材的内容方向是验证过的**。您自己盲测100条，95条完全跑不动；买我们验证过的素材，假设跨账户跑量率能到15-20%，已经是2-3倍提升了。而且价格比您找达人拍便宜，测试性价比更高。"

**Q2：素材的投放数据是真的吗？谁验证的？**

A2："MVP阶段，数据是我们人工从创作者千川后台截图验证的（会打码敏感信息）。后续如果拿到巨量ISV资质，可以直接对接千川API，数据100%真实。您如果不放心，可以先买1-2条测试，跑出效果再批量采购。"

**Q3：买了素材不跑量怎么办？**

A3："试点阶段，我们提供'7天消耗保障'——如果7天内消耗不足1000元（说明系统根本不给量），退50%。但我得说清楚，跑量是多因素的（素材+出价+人群+账户权重），我们只能保证素材本身是验证过的，不能保证在您账户上一定跑出同样ROAS。"

**Q4：价格能便宜点吗？**

A4："试点阶段首条5折，美妆素材400-750元。您想一下，这个价格比找腰部达人拍（1-2万）便宜10倍以上，比代拍机构（1000-3000）也便宜，而且是已验证过能跑的。如果您批量采购（一次买10条以上），我们可以再给8折。"

**Q5：素材版权/肖像权有问题吗？**

A5："所有素材都是创作者授权的，我们会签'素材使用授权协议'，明确您可以用于千川投流，授权期1年。肖像权方面，创作者出镜的素材，我们会要求创作者签署肖像授权书。如果后续有纠纷，我们平台负责处理。"

**Q6：我能修改素材吗（换产品/价格）？**

A6："买断后您可以二次剪辑，但有个前提——**不能改变素材的核心结构**（因为这个结构是验证过能跑的）。比如您可以替换产品特写镜头、修改价格字幕，但不能重新配音、换背景音乐、大幅度改剪辑节奏。我们会提供'源文件+跑量结构说明'，告诉您哪些部分可以改。"

**Q7：这个跟"有米有数"看竞品素材有什么区别？**

A7："有米有数只能看到竞品在投什么，但看不到真实的跑量数据（CTR/CVR/ROAS），而且看到了还得自己翻拍。我们是直接给您源文件+真实投放数据，买了就能用。相当于有米有数是'望远镜'，我们是'武器库'。"

**Q8：你们会不会把我的素材也拿去卖给别人？**

A8："不会。我们只收录创作者主动授权的素材，不会碰商家的自有素材。而且我们卖的素材会标注'已授权给X个商家使用'，您可以选择买独家（价格更贵）或非独家（同一条素材最多卖给5个商家）。"

**Q9：我先用着，跑出效果再付钱行吗？**

A9："不行。素材是买断制，一手交钱一手交货。但我们有'7天消耗保障'——跑不动退50%。如果您担心风险,建议先买1-2条测试，确认效果后再批量采购。"

**Q10：你们这个会不会被抖音/千川封禁？**

A10："我们走的是巨量ISV合规路径，不触碰平台核心利益（我们不做流量分发，只做素材工具）。而且素材交易本身是合法的（类似星图素材集市），我们是帮创作者和商家撮合交易。后续会申请ISV资质，接入千川生态。"

## 4. 试点方案

### 给商家的具体试点条件

**方案类型：限时优惠试点（非免费）**

| 项目 | 内容 |
|------|------|
| 试点优惠 | 首条素材5折，后续9折 |
| 采购数量 | 建议首次采购3-5条（批量测试更有代表性） |
| 支付方式 | 微信/支付宝转账，收到款后24小时内交付素材 |
| 交付内容 | 高清源文件（MP4，1080P以上）+ 素材跑量数据截图 + 跑量结构说明文档 |

**为什么不免费：**

1. 免费会降低商家的重视度（拿回去不一定用）
2. 免费无法验证"商家的真实支付意愿"（这是Gate 1的关键指标）
3. 5折已经是很大的优惠（美妆素材400-750元 vs 代拍1000-3000元）

**商家需要配合什么：**

| 配合项 | 具体要求 | 为什么需要 |
|--------|----------|-----------|
| 投放测试 | 将采购的素材上传千川投放至少7天 | 验证H2和H4（跨账户跑量效果） |
| 数据分享 | 投放7天后提供关键数据：消耗金额、CTR、CVR、ROAS | Gate 1决策依据 |
| 反馈沟通 | 投放过程中遇到问题（如素材不过审）及时反馈 | 优化产品和流程 |
| 时间配合 | Week 3完成采购，Week 4完成投放和数据回传 | 确保Gate 1如期决策 |

**我们承诺什么：**

| 承诺项 | 具体内容 |
|--------|----------|
| 素材质量 | 所有素材都经过人工审核，确保清晰度、版权合规 |
| 数据真实性 | 投放数据来自创作者千川后台截图（脱敏处理） |
| 消耗保障 | 7天内消耗不足1000元，退50%（需提供千川后台截图） |
| 快速交付 | 收款后24小时内交付源文件 |
| 1v1指导 | 试点期间提供投放指导（如何设置定向、出价建议） |

**我们不承诺什么：**

| 不承诺项 | 说明 |
|----------|------|
| 不承诺ROAS | 跑量效果受多因素影响（账户权重/出价/人群/竞价环境），我们只承诺素材本身是验证过的 |
| 不承诺跑量时长 | 素材生命周期通常7-14天，但具体多久衰减取决于账户 |
| 不承诺过审 | 千川审核规则复杂，如素材不过审，我们协助修改或更换同类素材 |

**退出机制：**

| 场景 | 退出方式 |
|------|----------|
| 素材7天消耗<1000元 | 提供千川后台截图，退50% |
| 素材不过审且无法修改 | 更换同价值素材或全额退款 |
| 对素材不满意（主观） | 购买后24小时内未使用可退款（扣10%手续费） |
| 素材有版权纠纷 | 平台承担法律责任，全额退款+赔偿损失 |

## 5. 商家筛选标准

### 必须条件（缺一不可）

| 条件 | 验证方式 |
|------|----------|
| 在投千川 | 要求提供千川后台截图（可打码敏感信息） |
| 月投放≥30万 | 从后台截图验证，或要求提供近30天消耗曲线 |
| 有专职投手 | 对接人必须是实际操盘投手（问3个专业问题即可判断） |
| 品类匹配 | 美妆/食品/3C三选一 |
| 数据配合意愿 | 明确同意投放后提供数据 |

### 排除条件（有一条立即排除）

| 排除项 | 原因 |
|--------|------|
| 新账户（开户<3个月） | 账户权重低，跨账户迁移效果无法验证 |
| 投放数据极不稳定 | 无法建立baseline，测不出素材的增量价值 |
| 决策链条超过3层 | 试点周期会拉长，影响Gate 1时间 |
| 主要靠自然流量带货 | 对千川投流不够重视，配合度低 |
| 有严重版权纠纷历史 | 风险高 |
| 纯代运营公司（无自有品牌） | 数据真实性难验证，且品类杂 |

### 评分卡（对候选商家打分排序）

用于在25个候选商家中选出最优15-20个：

| 维度 | 分值 | 评分标准 |
|------|------|----------|
| **投放规模** | 0-20分 | 30-50万(10分)，50-100万(15分)，100-150万(20分) |
| **素材痛点强度** | 0-20分 | 每周测试<30条(5分)，30-60条(12分)，60-100条(20分) |
| **决策效率** | 0-15分 | 投手无自主权(0分)，3000以下自主(10分)，5000以下自主(15分) |
| **数据配合意愿** | 0-15分 | 不愿意(0分)，可以但有顾虑(8分)，积极配合(15分) |
| **品类优先级** | 0-15分 | 美妆(15分)，食品(12分)，3C(10分) |
| **账户稳定性** | 0-10分 | ROAS波动>50%(0分)，波动30-50%(5分)，波动<30%(10分) |
| **沟通顺畅度** | 0-5分 | 响应慢/专业度低(0分)，正常(3分)，响应快/专业度高(5分) |

**总分100分，优先选择总分≥65分的商家进入试点。**

**优先级排序规则：**

1. 总分≥75分：第一优先级，Week 2立即深度沟通
2. 总分65-74分：第二优先级，根据Week 2进展决定是否启动
3. 总分<65分：备选，只有前两档不够时才考虑

## 总结：首周执行的核心要点

1. **Day 1-2最关键**——社群渗透+服务商对接必须快速启动，否则后面没有足够的候选商家

5. **保持弹性**——如果某个渠道效果特别好（如服务商推荐转化率超预期），立即加大投入
