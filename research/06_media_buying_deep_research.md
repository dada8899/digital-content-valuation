# 千川投放实战深度研究：需求侧真实情况

> 视角：日操盘百万级预算的千川广告投手
> 目的：把"投放"这件事的底层逻辑彻底拆开，帮团队理解需求侧的真实情况
> 日期：2026-02-07

---

## 1. "跑量"的真实解剖

### 1.1 "跑量"到底意味着什么

投手嘴里的"跑量"不是一个模糊的感觉，而是一个非常具体的状态：**一条计划在设定的出价范围内，能持续稳定地消耗预算，且转化成本不崩。**

更精确地说：
- "跑量"= 系统愿意给你曝光（过了冷启动）+ 用户愿意点（CTR达标）+ 点了之后愿意买（CVR达标）+ 这三件事能持续至少6-24小时
- "不跑量"= 上面任何一个环节断了。最常见的是第一步——系统根本不给曝光，计划"空耗"

行业真实数据：一个成熟账户每周上新50-100条素材计划，最终能"跑量"的通常只有3-8条，跑量率不到10%。这就是投手每天的核心焦虑来源。

### 1.2 跑量的完整因果链

一条素材从上传到"跑量成功"，经历以下因果链：

**第一关：系统预估（决定是否给曝光）**
- 素材上传后，千川的模型会基于素材特征（封面、前3秒、文案、历史相似素材表现）给出一个预估eCPM
- 这个预估eCPM × 你的出价 = 你在竞价池中的竞争力
- 如果预估值太低，系统直接不给曝光机会，计划"冷启动失败"
- 决定因素：素材的视觉吸引力（前3秒）、账户历史权重、出价竞争力、当前竞价环境

**第二关：用户是否点击（CTR）**
- 曝光到了用户面前，用户在0.5-1.5秒内决定是否停留
- 决定因素：封面/首帧（占50%）、前3秒hook（占30%）、与用户兴趣的匹配度（占20%）
- 千川信息流广告的平均CTR在2-5%之间，优质素材可到8-12%

**第三关：用户是否转化（CVR）**
- 点进来之后，用户需要完成目标动作（下单/留资/加购）
- 决定因素：产品本身（占40%）、素材的说服力/信任感（占25%）、落地页/直播间承接（占20%）、价格/促销力度（占15%）
- 这一步"内容"的权重其实在下降，产品力和价格力在上升

**第四关：系统持续放量（正反馈循环）**
- 前10-50个转化（冷启动期）的数据好 → 系统提高预估eCPM → 给更多曝光 → 更多数据 → 正反馈
- 如果冷启动期数据不好 → 系统降低预估 → 减少曝光 → 计划"跑死"
- 这一步决定了一条素材是"闪了一下"还是"真正跑起来"

### 1.3 各因素权重的经验估算

基于日常操盘的体感和数据复盘，我给出以下权重分配：

| 因素 | 权重 | 解释 |
|------|------|------|
| **素材内容质量** | 30-35% | 包括前3秒hook、整体叙事、视觉质量、信任感。这是投手最能主动干预的变量 |
| **人群定向** | 15-20% | 千川智能放量时代，精细定向的重要性在下降。系统探索能力已经很强，人为限制反而可能限制跑量。但冷启动期精准定向仍然关键 |
| **出价策略** | 15-20% | 出价太低不跑量，出价太高亏钱。核心是找到"刚好能跑"的临界点。控成本 vs 放量 vs 最大转化，策略选择影响很大 |
| **账户历史/权重** | 15-20% | 同一条素材，老账户跑量概率显著高于新账户。系统对账户有"信任分"，历史转化好的账户，冷启动期更短、起量更快 |
| **时间节点/竞价环境** | 5-10% | 大促前竞价激烈，成本飙升；凌晨竞价松，容易跑量但人群质量差。同样的素材，周二和周五的表现可能差30% |
| **产品本身** | 10-15% | 产品好不好卖、价格是否有竞争力、是否应季，这些投手无法改变但深刻影响跑量 |

**注意：以上权重之和超过100%，因为这些因素是交互的，不是简单叠加。** 一条90分的素材配上20分的产品，可能跑不过60分的素材配上90分的产品。

### 1.4 换账户投放的跑量衰减

**真实情况：同一条素材换账户，跑量概率通常下降30-60%。**

下降的原因拆解：
1. **账户模型差异（占衰减的40%）**：每个千川账户背后有一个独立的用户画像模型。A账户跑美妆跑了6个月，系统对"什么样的用户会买美妆"已经有很精准的判断。把同一条素材放到B账户（新号或跑别的品类的号），系统需要重新学习，冷启动期变长，成功率降低
2. **人群包不同（占衰减的25%）**：即使同品类，A账户积累的种子人群和B账户不一样，系统基于不同种子人群探索出的流量池也不同
3. **账户权重/信任分（占衰减的20%）**：新账户或历史数据差的账户，系统给的初始曝光机会更少，冷启动更难
4. **竞价环境变化（占衰减的15%）**：换账户通常不是同一时间投放，竞价环境已经变了

**但这不意味着跨账户迁移没有价值。** 关键点：一条在A账户ROAS 3.0的素材，即使在B账户衰减到ROAS 2.0，它也大概率比B账户自己从零测出来的素材好。因为它至少验证了"内容本身是有吸引力的"，衰减主要来自账户侧而非内容侧。

## 2. 投手的素材获取现状

### 2.1 所有渠道逐一列举

**渠道一：内部团队自制**
- 做法：公司养拍摄团队（编导+摄影+剪辑），日产10-30条
- 优势：成本可控（月薪制）、可快速响应需求、了解产品
- 劣势：创意枯竭快（同一个团队3个月后素材同质化严重）、风格单一、产量受人力限制
- 成本：团队月成本8-15万（3-5人），分摊到单条素材成本约300-800元
- 效率：跑量率5-8%

**渠道二：外包代拍机构**
- 做法：找视频代拍公司，按条/按月结算
- 优势：产能弹性大、可选不同风格、不用管团队
- 劣势：不了解产品（需要反复沟通）、质量波动大（好的代拍机构档期满）、交付周期7-15天
- 成本：单条500-3000元（看品类和质量要求），美妆品类通常1000-2000元/条
- 效率：跑量率3-6%（通常比自制还低，因为不够了解产品）

**渠道三：达人合作/星图**
- 做法：通过星图或私下联系达人，达人出镜拍摄
- 优势：达人自带信任感、内容质量通常较高、有真实使用场景
- 劣势：**贵**（达人费1万-50万不等）、沟通周期长（10-20天）、一次合作只产出1-3条素材、效果不可控
- 成本：单次合作1-5万（腰部达人），头部达人10万+
- 效率：跑量率8-15%（确实比自制高，但性价比不一定好）

**渠道四：竞品素材"借鉴"**
- 做法：用有米有数/蝉妈妈等工具监控竞品跑量素材，然后"翻拍"
- 优势：方向确定性高（别人跑出来了说明内容方向对）、决策成本低
- 劣势：**只能看到外在形式看不到数据**、翻拍总是差一点、全行业都在抄导致同质化、涉及合规风险
- 成本：工具年费2-5万 + 翻拍制作成本
- 效率：跑量率6-10%（方向对但执行有损耗）

**渠道五：巨量创意/即创等官方工具**
- 做法：用平台提供的模板工具、AI生成工具
- 优势：免费/低成本、产出快、模板有一定跑量基础
- 劣势：**同质化极其严重**（几十万投手用同样的模板）、AI生成素材目前转化率偏低、缺乏真实感
- 成本：几乎为零
- 效率：跑量率2-4%（量大但质低）

**渠道六：素材混剪/二创**
- 做法：把历史跑量素材拆解，重新混剪、加新元素
- 优势：成本极低、速度快、基于已验证的素材
- 劣势：本质是"吃老本"、系统会识别高度相似素材降权、生命周期短（3-5天）
- 成本：剪辑人力成本，单条约50-200元
- 效率：跑量率4-7%

### 2.2 如果有一个"素材市场"，它要提供什么我才会用

不说"好素材"这种废话。具体到：

**必须有的：**

1. **可见的投放数据**：不是播放量、点赞数这种自然流量数据，而是**千川投流数据**——消耗金额、eCPM、CTR、CVR、ROAS。这些数据是判断素材是否真正"跑量"的唯一依据。没有这些数据的素材市场，跟图片素材网站没区别
2. **品类匹配**：我投美妆，你给我看3C素材没用。必须能按投放品类精确筛选，而且品类粒度要细到"面膜""精华""口红"这一级，不能只有"美妆"
3. **可直接投放的源文件**：不是成品视频链接，而是高清源文件（最好带分轨），拿到就能上传千川开投
4. **价格合理且确定**：明码标价，不要议价。价格必须低于我找达人拍摄的成本，且低于预期消耗的一定比例（后面详细算）
5. **合规授权清晰**：肖像权、音乐版权、品牌元素授权一次性解决，不要投放中途收到律师函

**最好有的：**

6. **素材的"适用场景"标签**：这条素材适合什么品类、什么客单价区间、什么投放目标（直播间引流/短视频成交/商品卡）
7. **相似素材推荐**：我看中了一条，给我推10条类似的，让我同时测试
8. **效果保障机制**：买了不跑量怎么办？不要求全额退款，但至少有某种保底或补偿

### 2.3 最想要但现在怎么都得不到的素材

**真人口播 + 已验证跑量 + 可更换产品信息的"模板型"素材。**

具体来说：
- 一个真人（不是达人，就是普通好看/有信任感的人）对着镜头讲产品
- 这条素材在别的账户上已经验证过能跑（ROAS 2.0+）
- 我可以只替换其中的产品相关信息（产品名称、价格、促销信息），保留整体结构和人物
- 合法合规，不会有肖像权纠纷

**为什么得不到：**
1. 有跑量数据的素材，商家自己还在用，不可能给别人
2. 素材过期后（7-14天），数据也随之沉没，没有系统性地收集和流通
3. 真人口播涉及肖像权，个人合规成本太高
4. 目前没有任何渠道能同时解决"已验证"+"可复用"+"合规"这三个问题

## 3. "确定性"的精确定义

### 3.1 投手说的"确定性"到底是什么

三层含义，从表到里：

**第一层（表面）："这条素材大概率不会亏"**
- 投手每天要做的核心决策是"今天测哪些素材"。每测一条素材就是在花钱赌博——冷启动期的消耗（通常500-2000元）如果没跑出来，就是纯亏损
- "确定性"首先意味着**降低这笔测试成本的风险**。一条有历史数据的素材，至少告诉我"方向是对的"，我的测试预算不太可能完全打水漂

**第二层（中层）："我能跟老板交代"**
- 投手不是花自己的钱，是花公司的钱。每周测50-100条素材，其中90%以上跑不动，老板看到的是"每周花了5-10万测试费，只跑出来3条"
- "确定性"意味着**决策的可解释性**。我买了一条"别的账户ROAS 2.5"的素材来测，即使这条最终没跑起来，我也能说"我的决策逻辑是对的，是账户/时间等因素导致的"，这比"我自己拍了一条凭感觉应该能跑"要好交代得多

**第三层（底层）："我能稳定产出，不焦虑"**
- 投手最大的职业焦虑是"素材荒"——上周跑量的素材这周衰退了，新素材还没测出来，消耗在掉，ROI在跌
- "确定性"的终极含义是**稳定的素材供给管道**。不是一条素材确定能跑，而是"我有一个可靠的渠道，能持续给我提供有一定跑量概率的素材，让我的投放不断粮"

### 3.2 现有体系中，什么东西提供了确定性

| 确定性来源 | 可靠度 | 限制 |
|-----------|--------|------|
| **本账户历史跑量数据** | 最高（9/10） | 只反映过去，素材衰退后无用 |
| **竞品监测工具看到的跑量素材** | 中高（7/10） | 只能看到"别人在投"，看不到效果数据；翻拍有损耗 |
| **同品类的"万能模板"**（如美妆的"三段式口播"） | 中（6/10） | 大家都在用，竞争激烈，效果递减 |
| **代拍机构的"案例库"** | 中低（5/10） | 案例可能注水，不同品跑量差异大 |
| **行业经验/直觉** | 因人而异（3-8/10） | 不可规模化，新手没有 |
| **平台官方模板** | 低（3/10） | 同质化太严重 |

### 3.3 "另一个账户ROAS 2.5"的历史数据，你会多信任它

**信任度：6/10。会认真看，但不会全信。**

**信的部分：**
- 说明这条素材的**内容方向是对的**——前3秒hook有效、叙事逻辑通顺、用户看了会有购买欲
- 说明这条素材**至少过了系统的冷启动门槛**——eCPM预估不低
- 这两点已经比"从零猜测"强很多了。如果我自己测素材跑量率是5%，有ROAS数据背书的素材跑量率如果能到15-20%，那就有价值

**不信的部分：**
- ROAS 2.5是**那个账户**的ROAS，不是我的。他的账户模型、人群积累、出价策略、产品客单价可能跟我完全不同
- 素材的**时效性**：他跑出ROAS 2.5是什么时候的？上周？上个月？竞价环境变化很快，一个月前的数据参考价值有限
- **数据真实性**：谁来保证这个ROAS 2.5是真的？如果是平台提供并背书的数据，我信8分；如果是创作者/商家自己报的，我信3分
- **品类匹配度**：他卖面膜卖出ROAS 2.5，我卖精华液能复制吗？同是美妆，细分品类差异也很大

**底线：** 如果是平台认证的数据（来自千川后台，不可伪造），且匹配我的投放品类，且数据是近7天内的——我会把它作为"优先测试"的素材，愿意为它多付一点钱。但我绝不会因为这个数据就"闭眼买入"，它最终还是要在我的账户上跑出真实数据才算数。

## 4. 你会买什么？不会买什么？

### 4.1 五个"我一定会买"的场景

**场景一：美妆口播素材，有3天内千川ROAS数据，价格800-1500元**
- 品类：美妆（面膜/精华/粉底液）
- 形式：真人口播30-60秒，有产品展示镜头
- 数据：近3天在同品类账户跑出ROAS 2.0+，消耗超过5万
- 价格：800-1500元
- 保障：平台数据认证 + 7天内跑不量可申请50%退款
- 为什么一定买：**我在美妆品类每周测试费就花2-3万，如果花1500买一条概率更高的素材，替代5-10条盲测的素材，净省至少3000-5000元**

**场景二：食品类产品展示素材包（5条），价格500-1000元/包**
- 品类：食品（零食/饮品/预制菜）
- 形式：产品拆箱+使用场景+口播混剪包，5条不同角度
- 数据：包内至少2条有千川跑量记录
- 价格：500-1000元/包（单条100-200元）
- 保障：源文件交付，可二次剪辑
- 为什么一定买：**食品素材的跑量模板比较标准化，可迁移性强。5条一包的形式降低了单条风险**

**场景三：直播间引流素材，适配指定直播间风格，价格300-600元**
- 品类：任何在做千川直播投流的品类
- 形式：15-30秒的高能引流短片（不是完整内容，是feed流钩子）
- 数据：有历史引流率数据（点击→进入直播间的比例>5%）
- 价格：300-600元
- 保障：可预览完整素材再决定购买
- 为什么一定买：**引流素材是消耗量最大的素材类型，每天需要5-20条，且模板化程度高，跨账户迁移性好**

**场景四：3C数码评测风格素材，有详细拆解镜头，价格2000-4000元**
- 品类：3C数码（手机壳/充电器/耳机/小家电）
- 形式：专业评测风格，有产品对比、功能演示、使用场景
- 数据：同品类千川投放ROAS 1.8+
- 价格：2000-4000元
- 保障：含肖像授权+品牌元素可替换
- 为什么一定买：**3C评测类素材制作门槛高、成本高（自己拍要5000+），且真实感对转化率影响极大**

**场景五："跑量模板"——去掉产品信息的结构化素材框架，价格200-500元**
- 品类：通用（指定大类即可）
- 形式：带场景、带真人、带叙事结构，但产品信息区域留空/可替换
- 数据：原版（含产品信息版本）有跑量记录
- 价格：200-500元
- 保障：附带"跑量结构分析"（前3秒hook方式、转折点、CTA位置等）
- 为什么一定买：**这本质上是在卖"跑量的结构"，不是卖内容本身。我最需要的其实就是这种经过验证的叙事结构**

### 4.2 五个"打死不买"的场景

**场景一：没有任何投放数据的"优质素材"**
- 不管拍得多精美、制作多精良，如果没有千川投放数据证明它能跑量，对我来说就是"漂亮的废物"
- 自然流量数据（播放量、点赞）跟投流效果是两回事。10万播放的视频投千川可能完全跑不动

**场景二：头部达人的"过期内容"，要价5000+**
- 头部达人的内容价值高度绑定其个人IP，脱离了那个达人、那个账户、那个粉丝群体，内容本身的投流价值大打折扣
- 5000元我可以找腰部达人定制一条全新的素材，为什么要买过期的

**场景三：纯AI生成的"素材"**
- 2026年的AI素材在千川上的转化率仍然明显低于真人素材（大约低30-50%）
- 用户对AI面孔的识别能力在快速提升，恐怖谷效应真实存在
- 而且AI素材我自己用即创就能免费生成，为什么要花钱买

**场景四：跨品类的"万能素材"**
- 有人告诉我"这条素材在母婴品类ROAS 3.0，你拿去投美妆也行"——不行
- 品类之间的人群差异、卖点逻辑、内容风格差异太大，跨品类迁移的成功率极低

**场景五：附带复杂授权条款的素材（限制使用时长/限制修改/限制投放渠道）**
- 千川投放节奏非常快，一条素材可能今天跑量明天就衰退
- 如果买来还要操心"这条素材授权还有3天就到期""这条素材不能修改""这条素材不能在快手投"——管理成本太高
- 我需要的是**买断、无限制、拿来就用**

### 4.3 愿意为一条素材付多少钱——算账逻辑

投手买素材的心理账本非常简单：

```
素材价值上限 = 预期消耗 × (1 - 1/预期ROAS) × 愿意分享的利润比例

举例：
- 预期消耗：5万（一条好素材在千川上的生命周期消耗）
- 预期ROAS：2.5（投1块钱赚2.5块，利润率60%）
- 利润：5万 × 60% = 3万
- 愿意分享给素材的比例：5-10%（因为跑量是多因素的，素材只是其中之一）
- 素材价值上限：3万 × 5-10% = 1500-3000元
```

**实际出价会比上限低很多**，因为：
1. 不确定这条素材在我账户上能不能跑出同样的效果（打7折）
2. 不确定素材的生命周期还剩多少（打5-8折）
3. 市场上有替代品（自制/代拍/翻拍）

**所以真实的支付意愿：**

| 品类 | 单条支付意愿 | 前提条件 |
|------|------------|----------|
| 美妆 | 500-2000元 | 有近7天跑量数据，ROAS 2.0+ |
| 食品 | 200-800元 | 有跑量数据，可二次剪辑 |
| 3C数码 | 1000-3500元 | 有跑量数据，含肖像授权 |
| 服饰 | 300-1000元 | 有跑量数据，风格匹配 |
| 教育/知识付费 | 500-2000元 | 有表单留资数据 |

**如果没有跑量数据，以上价格全部打3折。** 这就是"确定性"的货币价值——有数据和没数据，价格差3-5倍。

## 5. 竞品和替代方案

### 5.1 现在在用的工具/平台

**有米有数**
- 在用，年费约3万
- 解决了什么：能看到竞品在投什么素材、投了多久、预估消耗量级
- 没解决什么：**只能看到"外观"，看不到真实投放数据（CTR/CVR/ROAS都是猜的）**；看到了也只能翻拍，不能直接买
- 本质上是一个"竞品监测工具"，不是"素材获取工具"

**蝉妈妈/飞瓜数据**
- 在用，年费约2万
- 解决了什么：达人带货数据、直播间数据分析、热门商品追踪
- 没解决什么：偏向"数据分析"而非"素材获取"，不能直接买到素材

**巨量创意/即创**
- 免费使用
- 解决了什么：AI辅助生成素材、提供模板、批量制作
- 没解决什么：**生成质量还不够好，同质化严重**，所有投手用同一套模板，效果在快速递减

**素材代拍机构（星辰传媒、闪电传媒等）**
- 按需合作，单条500-3000元
- 解决了什么：解放内部团队产能、可选不同风格
- 没解决什么：**交付周期长（7-15天）、不了解产品、没有效果保障、质量波动大**

**星图素材集市**
- 偶尔用
- 解决了什么：可以直接购买达人授权素材，合规有保障
- 没解决什么：**没有跑量数据、价格偏高（达人定价，不是市场定价）、供给稀少（大部分达人没上架）、品类覆盖不全**

### 5.2 "内容交易市场"的独特价值是什么

跟以上所有工具/平台比，真正的差异化只有一个：

**"带有真实投放效果数据的素材直接交易。"**

拆开说：
- **有米有数**给你"别人在投什么"→ 我们给你"别人投了效果是什么 + 你可以直接买"
- **代拍机构**给你"我们觉得能跑的素材"→ 我们给你"已经跑出来了的素材"
- **巨量创意**给你"AI认为能跑的模板"→ 我们给你"真人真实跑出来的数据"
- **星图素材集市**给你"达人觉得值这个价的素材"→ 我们给你"市场验证了值这个价的素材"

**核心差异：从"猜测能跑"到"验证已跑"。**

如果做不到"带真实投放数据"这一点，这个市场就只是另一个素材代拍平台，没有独特价值。

## 6. 一个关键的诚实回答

### 6.1 如果明天有这个工具，我会用吗

**会用。但不会是主力工具，而是"补充渠道"。**

使用频率：每周看1-2次，每次花10-20分钟浏览，每月买5-15条素材。

原因：
- 我的素材获取主力仍然是内部团队自制（因为最了解产品、响应最快）
- 但每周总有2-3天"素材荒"的时候，自制的素材测完了、新的还没出来、消耗在掉——**这个时候我会上去找"救急"的素材**
- 另外，每当进入一个新品类（比如公司新上了一条产品线），我会大量采购同品类的已验证素材来**快速了解"这个品类什么样的内容能跑"**，这比自己从零摸索效率高10倍

### 6.2 身边的投手朋友会用吗

**两极分化：**

- **会用的（约30-40%）**：操盘预算大（月耗50万+）、对素材有持续焦虑、愿意尝试新工具的投手。这些人素材需求量大，任何能提高跑量率的工具都愿意试
- **不会用的（约60-70%）**：以下几类——
  - 预算小的（月耗10万以下）：买素材的钱不如多测几条自制的
  - 依赖固定团队的：公司有成熟的内部制作流程，不需要外部素材
  - 不信任外部素材的："别人跑出来的素材在我这跑不动"这个认知很顽固
  - 习惯"抄"的：觉得看竞品翻拍就够了，为什么要花钱买

### 6.3 如果大部分人不会用，真正的原因是什么

说到底，核心原因有三个：

**原因一：信任问题没解决（占40%）**
- "ROAS 2.5"这个数据是真的吗？谁验证的？
- 如果是平台方提供的经过审计的数据，信任度高很多。如果只是卖方自报，基本不信
- **这个问题不解决，市场就是空谈**

**原因二：迁移效果的不确定性（占30%）**
- 即使数据是真的，投手心里很清楚：别人账户跑出来的数据，在我这里要打折
- 这个折扣有多大？没有人知道。这种"不确定性中的不确定性"让投手倾向于保守——不如花同样的钱多测几条自制素材
- **如果能建立"跨账户效果预测模型"（比如：基于你的账户特征，这条素材在你这里的预估ROAS是X），信任度会大幅提升**

**原因三：使用习惯的惯性（占30%）**
- 投手的工作流已经形成：每天打开千川后台 → 看昨天数据 → 调整出价 → 让团队出新素材 → 上传测试
- "去另一个平台/工具浏览和购买素材"是一个新动作，需要打断现有流程
- **如果这个工具能嵌入千川后台（作为千川的一个功能模块），而不是独立存在，使用率会高得多**
- 这也是final_project_plan里"做千川生态内的工具，不做独立平台"这个决策正确的原因

## 总结：对项目方案的几点直接反馈

1. **核心假设H4（素材跑量效果可跨账户迁移）是最关键的。** 作为投手，我对此持"谨慎乐观"态度——方向可迁移，但效果数据会衰减30-60%。如果最终验证衰减在30%以内，这个市场有巨大价值；如果衰减超过50%，市场就很薄了

2. **价格区间（美妆800-2500，食品200-600）基本合理。** 但有一个细节：投手不是按"条"买的，而是按"测试批次"买的。如果能提供"品类素材包"（5-10条/包，打包价折扣），购买频率和客单价都会上去

3. **"跑量数据"必须是千川后台数据，不是自然流量数据。** 播放量100万的视频在千川投流可能完全跑不动。如果平台不能提供经认证的投流数据，就只是另一个素材网站

4. **嵌入千川是对的。** 独立做一定死。投手每天盯着千川后台8-12个小时，任何需要"跳出千川"才能用的工具，使用率都会非常低

5. **最该做的一件事：建立"素材-账户-效果"的三角数据集。** 同一条素材在不同账户的跑量效果对比——这个数据集目前全行业都没有，谁先有谁就有不可替代的定价和推荐能力
